# NFT MINT APP SAMPLE

This is an app that simulates the mint npt process

## Features

- Mint NFT
- Show balance
- Approve other account to mint nft

## Tech

##### - Blockchain

- Language: [Solidity]
- Framework: [Hardhat]

##### - Backend

- Language: [Javascript, NodeJS]
- Framework: [Express]

##### - Frontend

- Language: [Javascript]
- Framework: [ReactJS]

## Installation

#### Run at localhost

```sh
#open terminal 1
cd nft-contracts
npm i
npx hardhat node
#open terminal 2
cd nft-contracts
npx hardhat run --network localhost ./scripts/deploy.js
#open terminal 3
cd app
npm i
npm start
# install metamask walet
import private key of account owner to metamask and connect that account to  website.
```
